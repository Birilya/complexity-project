    # Задача о миниальном k-разрезе
    ## *Обозначения:*
    + $\textbf{Min k-Cut}$ $:=$ $\{\Big (G = (V, E) , w, C, k\Big ) | \space \exist \space \hat{E} \subset E \text{ стоимости} \le C, \text{что  граф }(V, E \backslash \hat{E}) \text{ распадается на} \ge \text{k компонент связности} \}$
    + $\textbf{Min Balanced Cut}$ $:=$ $\{\Big (G = (V, E) , B, w, C\Big ) | \space \exist \space \text{множества вершин } V_1, V_2: V = V_1 \cup V_2; |V_1|, |V_2| \le B \ и \ \text{сумма весов ребер между } V_1 \ и \  V_2 \le C\}$
    + $\textbf{Max Cut}$ $:=$ $\{\Big (G = (V, E), C\Big ) | \space \exist \space \text{множество вершин } \hat{V} , \text{что в графе G между V и } V \backslash \hat{V} \text{ хотя бы C ребер}\}$
    + $\textbf{NAE-3SAT}$ $:=$ $\{\text{множество 3-КНФ, у которых сущетсвует набор, что в каждом дизъюнкте формулы встречается как 0, так и 1} \}$

    ## *Договоренность*
    Для формальности будем считать $k \le |V|$ в $\text{Min k-Cut}$



    ## Пункт *a*:
    ### Доказательство принадлежности NP
    Предъявим сертификат $y$ и верификатор $V(x, y)$:

    * $y$ - набор ребер $\Big( \hat{E}, \textit{список вершин для каждой компоненты}\Big)$ - набор из $k + 1$ элемента
    * $V(x, y)$  = $\begin{cases} \text{Распаковывает x, y, где x} = (G, w, C) \\ \text{Проверяет, что сумма весов в }\hat{E} \le C \\ \text{Для каждой пары компонент проверяет, что между их вершинам нет ребер}     \end{cases}$

    #### Почему корретно?
    Корректность соотвествия $l \in L \Leftrightarrow \exist y \  V(x, y) = 1$ понятна по построению.

    Оценка сложности: $|G| = poly(|V|). \newline$ Нетрудно убедиться, что каждый шаг верификатора также выполняется за $poly(|V|)$:

    1. За $O(|G| + |E| + 1)$
    2. За $O(|E|)$
    3. За $O(k^2 \times |V| ^ 2) \le O(|V|^4)$

    ### Доказательство принадлежности NPC
    Докажим в три шага:
    $$\textbf{NAE-3SAT} \ \overset{(1)}{\le_p} \ \textbf{Max Cut} \ \overset{(2)}{\le_p} \ \textbf{Min Balanced Cut} \ \overset{(3)}{\le_p} \ \textbf{Min k-Cut}$$

    *Примечание: NP-полнота $\text{NAE-3SAT}$ доказывалась на лекции*

    *P.S. принадлежность NP задач Max Cut и Min Balanced Cut я опущу для краткости (к тому же понятно, какие верификаторы и сертификаты нужны ;)). Готов ответить устно на семинаре или дописать, если необходимо*

    #### Шаг 1
    Строим f: $x \in \textbf{NAE-3SAT} \Leftrightarrow f(x) \in \textbf{Max Cut}$

    Пусть на вход поступила $\phi$. Пусть $(x_1, x_2 \dots x_n) - переменные, (C_1, \dots C_m) - дизъюнкты$. Тогда 
    + V = \{$x_1 \neg x_1, x_2, \neg x_2, \dots x_n, \neg x_n $ \}
    + $E = E_1 \cup E_2$, где 
        * $E_1 = \{(x_i, \neg x_i) \ \forall i \in \{1, 2, \dots n\}\}$
        * $E_2 = \{\text{тройки ребер, соединяющие вершины из} C_i \ \forall i \in {1, 2, \dots n} \}$
    + $С = 2m + n$

    Доказательство корректности:

    $\Rightarrow:$
    $\newline \ \ \ \ $ Положим $\hat{V}$ - истинные литералы из искомого набора ([см обозначения](#обозначения)), $V \backslash \hat{V}$ - ложные.
    $\newline \ \ \ \ $ Т.к. в $\forall$ треугольнике будут как истинные, так ложные литералы $\Rightarrow$ ровно 2 его ребра будут находиться между $\hat{V}$ и $V \backslash \hat{V}$
    $\newline \ \ \ \ $ Также каждая пара $х, \neg x$ будет содержать ребро между  $\hat{V}$ и $V \backslash \hat{V}$
    $\newline \ \ \ \ $ Значит, всего ребер  между  $\hat{V}$ и $V \backslash \hat{V}$ $\ge 2m + n$

    $\Leftarrow:$
    $\newline \ \ \ \ $ Аналогично. Положим литералы $\hat{V}$ - истинные,  $V \backslash \hat{V}$ - ложные. Докажем, что такое определение корректно и полученные набор - искомый для $\phi$.

    $\newline \ \ \ \ $ Пойдем от противного. Пусть $\neg x_i$ и $x_i$ лежат в $\hat{V}$. Тогда всего ребер между  $\hat{V}$ и $V \backslash \hat{V} \le n - 1 + 2m \lt C$ , т.к. между половинами может быть не более 2 ребер от каждый треугольника и одного ребра из оставшихся пар $(x, \neg x)$ - противоречие. Это доказывает корректность определения набора 

    $\newline \ \ \ \ $ Аналогично доказывается, что построенный набор - искомый: если в какой-то тройке все литералы одинаковые $\Rightarrow$ полностью лежит в $\hat{V}$ или $V \backslash \hat{V} \Rightarrow$ ребер между $\hat{V}$ и $V \backslash \hat{V} \le 2(m - 1) + n \lt C$

    Полиномиальность $f$ будем считать очевидной ;)
    #### Шаг 2
    *В этом абзаце будет шляпка ($\ \  \hat{} \ $), будет значит "объект из Min Balanced Cut", без шляпки -  Max Cut*
    Строим f:

    На вход поступил $G$ и $C$. Тогда
    * Пусть $|V| = n$
    * $\hat{G} = K_{2 n}$ - полная клика на $2n$ вершинах
    * $w(e) = \begin{cases} 1, \text{если } e \in E \\ 2, \text{если }e \not\in E, \text{т.е. ребро было добавлено} \end{cases}$
    * $\hat{C} = 2n^2 - C$
    * $\hat{B} = n$

    Доказательство корректности:

    $\Rightarrow:$
    $\newline \ \ \ \ $ Рассмотрим разрез $(S, V \backslash S)$ из Max Cut. Распределим новые вершины по долям, дополнив их до размера n (к первой добавим $n - |S|$, ко второй $n - |V \backslash S|$).
    $\newline \ \ \ \ $ Нетрудно убедиться в том, что величина разреза $ \le 2n^2 - C = \hat{C}$

    $\Rightarrow:$
    $\newline \ \ \ \ $ Рассмотрим две доли размера n - $V_1, V_2$. Если между ними разрез стоимости $\le \hat{C} \Rightarrow$ разрез в исходном графе $\ge 2n^2 - \hat{C} \Rightarrow (V_1 \cap V, V_2 \cap V)$ - искомое разбиение Max Cut

    Полиномиальность f следует из $|\hat{G}| \le 4 \cdot |G|$


    #### Шаг 3
    *Min Balanced Cut - частный случай Min k-Cut*

    На вход поступили $G, C, B, w$. Тогда 
    * $f = id$  
    * $k = 2$

    $\Rightarrow:$
    $\newline \ \ \ \ $
    Разрез в Min Balanced Cut является искомым разрезом в Min 2-Cut. 

    $\Leftarrow:$
    $\newline \ \ \ \ $
    Разрез в Min 2-Cut является искомым разрезом в для Min Balanced Cut (условие на $|V_{1,2}| \le B$ выполнено по определению f). 

    #### Итог
    Доказали **Min k-Cut** $ \in NPC$
